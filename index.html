iARoC iRobot Create 2 Instructions


Step 1:

Clone and download the repository at this link and import it into Eclipse like normal.

Step 2:

In the Eclipse menu bar, click Help->Install New Software...

Step 3:

In the bar titled "work with:" paste the following address and click "add"
https://raw.githubusercontent.com/tsvetan-stoyanov/launchpi/master/org.launchpi.us/target/site/site.xml

Step 4:

In the window that pops up, type launchPi in the "name" box and click "ok."

Step 5:

Check the box next to "Uncatacorized" and click "next."

Step 6:

Agree to the terms and continue the installation. You'll need to restart Eclipse once the installation is complete.

Step 7:

Now that the launchPi plugin is installed, you can run your Eclipse code on the robot's Raspberry Pi.
To do this, right click on the project folder, go to "Run As...," and click "Run Configurations."

Step 8:

Double click the Rasperry Pi icon on the left.

Step 9:

Next to "Raspberry Pi System," click "New..."

Step 10:

In the pop-up box type "rpi1" replacing the '1' with the correct number of your robot.

Step 11:

Click that "main" tab above the "Raspberry Pi System" field and go to the "main class:" section.

Step 12:

Click "search" and select the CleverRobot.java class.

Step 13:

Click "Run" and select the CleverRobot.java class.

Step 14:

If promted for the Raspberry Pi's username and password, set them both to pi.

Step 15:

Wait about 10 seconds and your robot should start to beep. That means it is ready to run the code.
Press the "Clean" button to start your program.




*ALTERNATE INSTRUCTIONS*




Step 1:

Clone and download the repository at this link and import it into Eclipse like normal.

Step 2:

Write your code in the setup and loop methods.



Step 3:

Export your code as a runnable jar.

Under Launch Configurations, select the class containing the main method for your CleverRPiRobot project.
Under Export destination, select the location of your jar file. The desktop is probably the easiest location to find.
Then click finish.

Click OK

Click OK again

Step 4:

Open two terminal windows.

Navigate one of them to the directory that you saved the jar file. 
cd Desktop

Step 5:

Copy the jar file to the robot's Raspberry Pi.
scp CleverRPiRobot.jar pi@rpi7:
Change the '7' to the actual number of your robot.

Type 'yes' when prompted. This should only happen the first time.


If you get the following error, make sure that the Rasperry Pi is connected to the robot.


When prompted for the password, type in 'pi'.


If the transfer was successful, you should see this:


Step 6:

In the other terminal window, connect to the robot's Rasperry Pi. 
ssh pi@rpi7.local
Change the '7' to the actual number of your robot.


If successfully connected, it should look like this:


Step 7:

Once connected to the Pi, you can run the program. Be sure to use 'sudo'.
sudo java -jar CleverRPiRobot.jar


Step 8:

If the robot starts to beep, then it is ready to run your code. Press the 'clean' button on the robot to start the program.
